# Typography Tokens

This page documents all typography tokens used in the Labs Design System. See the `Typography` story for live examples and values.

- **Font sizes:** `--font-size-base`, `--font-size-lg`, etc.
- **Font weights:** `--font-weight-bold`, etc.
- **Line heights:** `--line-height-base`, etc.

## Usage

Use these tokens via CSS custom properties:

```css
font-size: var(--font-size-base);
font-weight: var(--font-weight-bold);
```

## Best Practices
- Use tokens for all typography, never hardcode values
- Reference the design system docs for scale and usage

See [DESIGN-SYSTEM.md](../../../../DOCS/DESIGN-SYSTEM.md) for the full system.
