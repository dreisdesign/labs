# Colors Tokens

This page documents all color tokens used in the Labs Design System. See the `Colors` and `Root Colors` stories for live swatches and values.

- **Colors:** Semantic and theme-aware tokens (e.g., `--color-primary`, `--color-accent`)
- **Root Colors:** Base palette tokens (e.g., `--color-surface-container`, `--color-success`)

## Usage

Use these tokens via CSS custom properties:

```css
background: var(--color-primary);
color: var(--color-on-surface-variant);
```

## Best Practices
- Use semantic tokens for UI elements (buttons, backgrounds, text)
- Use root tokens for foundational color values
