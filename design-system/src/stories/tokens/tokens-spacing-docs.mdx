# Spacing Tokens

This page documents all spacing tokens used in the Labs Design System. See the `Spacing` story for live examples and values.

- **Spacing scale:** `--space-xs`, `--space-sm`, `--space-md`, `--space-lg`, etc.

## Usage

Use these tokens via CSS custom properties:

```css
margin-bottom: var(--space-md);
padding: var(--space-lg);
```

## Best Practices
- Use tokens for all spacing, never hardcode values
- Reference the design system docs for scale and usage

See [DESIGN-SYSTEM.md](../../../../DOCS/DESIGN-SYSTEM.md) for the full system.
