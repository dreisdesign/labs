# Icon Only Button Pattern

Icon Only is a usage pattern, not a variant. Use a regular button variant (e.g. primary) and provide only a slotted icon with an `aria-label` for accessibility.

## Accessibility
- Always provide an `aria-label` describing the button action.
- Use the `icon` variant for correct styling.
